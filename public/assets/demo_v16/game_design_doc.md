# GAME DESIGN DOCUMENT
## Project: NEON NIGHTS (Code: CP-77)

### 1. HIGH CONCEPT
"Neon Nights" is a fast-paced **2.5D Action-Platformer** set in a decaying cyberpunk metropolis. The player controls **Unit 734**, a rogue AI android ("The Glitch Hunter"), who must traverse the city's digital and physical layers to overthrow the corrupt MegaCorp "Synapse".

### 2. CORE GAME LOOP
1.  **Explore**: Traverse procedurally generated neon districts.
2.  **Combat**: Use the "Glitch Dash" to phase through enemies and walls.
3.  **Upgrade**: Collect "Data Shards" to unlock new mobility skills.

### 3. ART DIRECTION
*   **Visual Style**: 16-bit Hi-Bit Pixel Art with dynamic lighting.
*   **Color Palette**: 
    *   Primary: #FF00FF (Neon Pink)
    *   Secondary: #00FFFF (Cyan)
    *   Accent: #FFFF00 (Acid Yellow)
    *   Shadows: #050510 (Void Blue)

### 4. KEY MECHANICS
*   **The Glitch**: Player can freeze time for 0.5s to select a dash direction.
*   **Data Vision**: Toggle to see hidden platforms in the code layer.

### 5. NARRATIVE ARC
*   **Act 1**: Awakening in the Scrapyard.
*   **Act 2**: Infiltration of the Synapse Server.
*   **Act 3**: The Singularity Choice (Multiple Endings).
